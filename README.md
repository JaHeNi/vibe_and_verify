# Vibe & Verify
A series of coding collaborative sessions for developing one's skillset in technical AI safety.

## Setup (NixOS)
Run "nix develop" or "nix --extra-experimental-features "nix-command flakes" develop" to create the shell for the project.

Create venv with "python -m venv .venv --system-site-packages"

Activate the environment with source .venv/bin/activate
